## 開発の背景

### 1.新規のお客様を増やし、ココイチの魅力を伝えたいという思い

私はカレーハウス CoCo 壱番屋でアルバイトをする中で、ココイチには多くのリピート客がいる一方で、新規のお客様をさらに獲得していく必要性を強く感じていました。実際に働く中で、常連のお客様が多く来店される一方で、初めて来店する方は想像よりも少ないという印象がありました。そこで、ココイチのカレーの魅力をより多くの人に知ってもらうためには、店舗での接客だけではなく、別の形でアプローチすることも重要ではないかと考えるようになりました。その一つの手段として、ココイチに興味を持つきっかけを提供できるアプリを自分で開発したいと思うようになりました。アプリを通じて、メニューの特徴や楽しみ方、カスタマイズ性など、ココイチならではの魅力を自然に伝えられる仕組みを作ることが目的となっております。

### 2.アプリ開発を通して、自分自身のスキルを高めたいという思い

もう一つの動機として、アプリ開発の経験を積み、自分のスキルを向上させたいという思いがありました。元々プログラミングやアプリ開発に興味がありましたが、実際に自分で企画し、設計し、実装まで行う経験は大きな成長につながると感じました。特に、実際の店舗で感じた課題を解決するためのアプリを作ることで、単に技術を学ぶだけでなく、「現場で見つけた課題を、技術でどう解決するか」という視点を養うことができると考えました。

## 機能紹介

ココイチカレーアプリケーションには**ココイチカレー診断**と**ココイチカレーシミュレーター**の二つの機能があります。

### 1. ココイチカレー診断

#### 1.1 概要

この機能の目的は、**いくつかの簡単な質問に答えるだけで、自分にあったカレーの組み合わせを知ることができる**ようにすることです。ココイチには、ベースとなるカレーの種類に加えて、豊富なトッピングが用意されています。しかし、その自由度の高さがかえって迷いにつながり、「何を頼めば良いのかわからない」と感じてしまうお客様も少なくありません。特に、初めてココイチを利用する新規のお客様にとっては、注文のハードルになることがあります。

そこで、このカレー診断では、いくつかの質問に答えるだけで、好みに合ったおすすめのカレーを提示できるようにしました。利用者の好みのベースカレーとトッピングを提示することで、ココイチの魅力である**カスタマイズの楽しさ**を直感的に体験してもらえる仕組みになっています。

また、ココイチのキャラクターである「ネコイチ」のイラストを使用し、画面全体をかわいく、楽しい雰囲気にデザインしました。**診断をしている時間そのものが一つの体験となり、ココイチに興味を持つきっかけ**を作れるよう工夫しています。**（公開しているアプリでは、著作権の関係上画像は白塗りとなっております）**

#### 1.2 提供している機能

この機能では、ココイチでおすすめのカレーを 約20問の質問に答えるだけで診断できるようになっています。質問は ステップ1 と ステップ2 の二つに分かれており、ユーザーの好みに合わせて最適な組み合わせを導き出します。

まず ステップ1 では、ユーザーの味の好みや辛さ、食べたい気分などの回答から、最も相性の良いベースカレー を決定します。
続いて ステップ2 では、ユーザーのより細かな好みを基に、相性の良いトッピングを一つ選び、さらにおすすめのトッピングを四つ提案します。

それぞれの質問には「はい」「いいえ」で答えてもらい、その回答に応じて各カレーやトッピングにスコアを加算していきます。最終的に、スコアが最も高かったものが診断結果として選ばれる仕組みになっています。

結果表示画面の下にある 「シミュレーターで遊ぶ」ボタン を押すと、診断で決定した「ベースカレー」「スコアが最も高かったトッピング」が初期状態として設定され、さらに おすすめトッピング4つも一覧で表示されるようになっています。これにより、診断結果をそのまま注文の参考にしたり、シミュレーター内で別の組み合わせを試したりと、楽しみながらカレー選びができるよう工夫しています。

### 2. ココイチカレーシミュレーター

#### 2.1 概要

この機能の目的は、**注文したいカレーをシミュレーションすることで、合計金額を直感的に把握できるようにすること**です。ココイチには豊富なトッピングが用意されており、ベースカレー、ライスの量、辛さ、トッピングを自由に組み合わせられるのが魅力です。しかし、その自由度の高さゆえに、特に**複数のトッピングを選ぶ場合は計算が複雑になり、最終的な金額が分かりにくくなってしまう**ことがあります。

また、家族で来店されるお客様の中には、「○円まで」とあらかじめ予算を決めて注文される方もおり、人数が増えるほど個々の注文金額を把握するのが難しくなるという課題もあります。

そこで、このシミュレーターでは、注文したい「ベースカレー」「ライスの量」「辛さ」「トッピング」をそれぞれボタンで簡単に選択でき、その場で合計金額が視覚的にわかりやすく表示されるようにしました。これにより、はじめてココイチを利用する方でも、**組み合わせによる金額の変化を直感的に理解**でき、安心して注文内容を決められるようになっています。

また、操作性を高めるために画面には分かりやすい画像を用いており、直感的に選択できるUIにしています。**（公開しているアプリでは、著作権の関係上画像は白塗りとなっております）**

#### 2.2 提供している機能

この機能では、「ベースカレー」「ライスの量」「辛さ」「トッピング」を、それぞれボタン操作で簡単に選択できるようになっています。画面構成は 「カレー選択画面」「トッピング選択画面」「結果表示画面」 の3つで構成されています。

まず、カレー選択画面 では、ベースカレー・ライスの量・辛さを選択します。その後、トッピング選択画面 で追加したいトッピングを自由に選ぶことができます。

最後の 結果表示画面 では、選択した内容を基に組み合わせたカレーを画像でわかりやすく表示し、視覚的に完成形がイメージできるようにしています。これにより、注文内容を具体的に想像しながら楽しめる UI になるよう工夫しています。**（公開しているアプリでは、著作権の関係上画像は白塗りとなっております）**
